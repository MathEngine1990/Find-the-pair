Пользовательское соглашение

Игра "Память: Найди пару"

1. Общие положения

Данное пользовательское соглашение регулирует использование игры "Память: Найди пару" (далее - Приложение).

2. Описание сервиса

Приложение представляет собой игру на развитие памяти, где пользователь ищет парные карты.

3. Сбор данных

Приложение может собирать следующие данные:

  • Идентификатор пользователя ВКонтакте
  • Имя и фото профиля (для персонализации)
  • Игровую статистику (результаты, уровни)

4. Использование данных

Собранные данные используются только для:

  • Обеспечения работы игры
  • Сохранения прогресса
  • Улучшения пользовательского опыта

5. Возрастные ограничения

Приложение не содержит контента, запрещенного для несовершеннолетних. Возрастное ограничение: 0+
